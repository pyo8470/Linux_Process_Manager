# Bash TUI Process Manager

터미널에서 **사용자 목록과 해당 사용자의 프로세스 목록을 탐색·관리**할 수 있는 Bash 기반 TUI(Text User Interface) 프로그램입니다. 방향키로 이동하며 프로세스를 선택하고, CPU 사용량 확인 및 프로세스 종료(kill)를 지원합니다.

---

## 주요 기능

* `/etc/passwd` 기반 **시스템 사용자 목록 표시**
* 선택한 사용자의 **프로세스 목록 조회** (`ps` 기반)
* 방향키를 이용한 **사용자/프로세스 선택 이동**
* 선택 프로세스의 **CPU 사용률 확인**
* 본인 소유 프로세스에 한해 **강제 종료(kill -9)**
* 권한 없는 종료 시 **권한 오류 화면 표시**
* ASCII 아트 기반 UI

---

## 실행 환경

* OS: Linux (Ubuntu 등)
* Shell: **bash**
* 필수 명령어:

  * `ps`
  * `cut`
  * `grep`
  * `sort`
  * `rev`
  * `whoami`

> ⚠️ `bash` 전용 스크립트입니다. `sh`로 실행하면 정상 동작하지 않습니다.

---

## 실행 방법

```bash
chmod +x process_manager.sh
./process_manager.sh
```

---

## 화면 구성

```
-NAME-----------------CMD--------------------PID-----STIME----
| 사용자명             | F/B 명령어            | PID   | STIME |
```

* **NAME**: 시스템 사용자 계정
* **CMD**: 프로세스 명령어
* **PID**: 프로세스 ID
* **STIME**: 프로세스 시작 시간
* **F / B**:

  * `F` : Foreground
  * `B` : Background

색상 표시:

* 🔴 사용자 선택
* 🟢 프로세스 선택

## 키 조작 방법

### 이동

| 키 | 동작            |
| - | ------------- |
| ↑ | 위로 이동         |
| ↓ | 아래로 이동        |
| → | 사용자 → 프로세스 선택 |
| ← | 프로세스 → 사용자 선택 |

### 명령

| 키              | 설명                         |
| -------------- | -------------------------- |
| `k` 또는 `Enter` | 선택한 프로세스 강제 종료 (`kill -9`) |
| `cp`           | 선택한 프로세스 CPU 사용률 표시        |
| `q` / `Q`      | 프로그램 종료                    |

---

## 동작 로직 요약

1. `/etc/passwd`에서 사용자 목록을 읽어 정렬
2. 선택된 사용자 기준으로 `ps`를 이용해:

   * 명령어
   * PID
   * 시작 시간
   * 상태(F/B)
   * CPU 사용률
     을 배열로 관리
3. 20개 단위로 페이징 처리
4. 방향키 입력에 따라 커서 및 스크롤 인덱스 갱신
5. 권한 검사 후 프로세스 종료 또는 오류 출력

---

## 권한 처리

* 본인(`whoami`)이 소유한 프로세스만 종료 가능
* 타 사용자 프로세스 종료 시 **NO PERMISSION 화면 출력**

---

## 주의사항

* `killall -9` 사용으로 **동일 이름의 모든 프로세스가 종료**될 수 있음
* 시스템 핵심 프로세스 종료 시 시스템 불안정 가능
* 학습 및 실습 목적 사용 권장
